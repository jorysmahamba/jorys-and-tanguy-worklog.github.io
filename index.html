Sheaf Cohomology & Spectral Sequences

Productive session working through Hartshorne Chapter III. Finally understood the geometric interpretation of higher cohomology groups!

Key insight: For a coherent sheaf \(\mathcal{F}\) on a projective scheme \(X\), we have the vanishing theorem:

\(H^i(X, \mathcal{F}(n)) = 0\) for \(i > 0\) and \(n \gg 0\)

This has deep implications for understanding the global sections functor. Also worked through the proof of Serre duality - the pairing between \(H^i(X, \mathcal{F})\) and \(H^{n-i}(X, \omega_X \otimes \mathcal{F}^‚à®)\) is starting to make sense.

Proximal Gradient Methods & ADMM

Implemented and analyzed convergence of proximal gradient descent for composite optimization problems. The key is splitting the objective:

\(\min_{x \in \mathbb{R}^n} f(x) + g(x)\)

where \(f\) is smooth with \(L\)-Lipschitz gradient and \(g\) is convex (possibly non-smooth)

Derived the convergence rate \(O(1/k)\) for the function values. Also started exploring accelerated versions using Nesterov momentum - the improvement to \(O(1/k^2)\) is remarkable!

# Proximal gradient implementation
def prox_grad(f_grad, prox_g, x0, step_size, max_iter=1000):
    x = x0.copy()
    for k in range(max_iter):
        grad = f_grad(x)
        x = prox_g(x - step_size * grad, step_size)
    return x

Tomorrow: diving into ADMM for distributed optimization. The connection to augmented Lagrangian methods is fascinating.

Schemes and Morphisms

Today's breakthrough: finally internalized why we need schemes instead of just varieties. The nilpotent elements capture infinitesimal information!

Worked through examples of non-reduced schemes. The canonical example:

\(\text{Spec}(k[x]/(x^2)) \hookrightarrow \text{Spec}(k[x])\)

This "fat point" remembers first-order deformation data. Connection to tangent spaces and deformation theory is becoming clearer.
